# Nasjonal Arkitektur: Kapabilitetsmapping

**Enkel oversikt med eksempler**  
*Mapping mellom nasjonale kapabiliteter og fellesressurser*

| Kapabilitet & Definisjon | Produkter & Ressurser |
| :--- | :--- |
| **Strategisk styring**<br>*Å sette retning for nasjonal arkitektur og realisere strategiske mål.* | <ul><li>Digitaliseringsrådet</li><li>SKATE</li><li>Medfinansieringsordningen</li><li>Stimulab</li><li>Digitaliseringsrundskrivet</li></ul> |
| **Juridisk samhandling**<br>*Å forvalte et helhetlig juridisk rammeverk for sikker digital samhandling.* | <ul><li>Ressurssenter for deling av data</li><li>Veileder for digitaliseringsvennlig regelverk</li><li>Oversikt over EU-regelverk</li></ul> |
| **Informasjonsforvaltning**<br>*Felles rammeverk så virksomheter kan utveksle data og beskrivelser.* | <ul><li>data.norge.no (fellesløsning)</li><li>Rammeverk for informasjonsforvaltning</li><li>Orden i eget hus</li><li>Nasjonal verktøykasse for deling</li></ul> |
| **Informasjonssikkerhet**<br>*Tilstrekkelig sikkerhet i tjenester i felles økosystem.* | <ul><li>sikkert.no</li><li>Maskinporten (fellesløsning)</li><li>NSM Grunnprinsipper</li><li>Stifinneren (internkontroll)</li></ul> |
| **Tillit**<br>*Autentisering og autorisasjon på tvers av tjenestekjeder.* | <ul><li>ID-porten (fellesløsning)</li><li>Maskinporten (fellesløsning)</li><li>eSignering (fellesløsning)</li><li>Altinn autorisasjon (fellesløsning)</li></ul> |
| **Datautveksling**<br>*Sikker og standardisert utveksling av data mellom aktører.* | <ul><li>eFormidling (fellesløsning)</li><li>FIKS-plattformen (fellesløsning)</li><li>Altinn formidling (fellesløsning)</li><li>Altinn events (fellesløsning)</li></ul> |
| **Tjenesteutvikling**<br>*Å utvikle sammenhengende digitale tjenester.* | <ul><li>Altinn Studio / 3.0 (fellesløsning)</li><li>Den triple diamanten</li><li>Tjenestedesign-metodikk</li><li>Strategisk fremsyn</li></ul> |
| **Sluttbrukertjenester**<br>*Sammenhengende brukeropplevelse gjennom standardiserte tjenester.* | <ul><li>Norge.no (fellesløsning)</li><li>Altinn.no (fellesløsning)</li><li>eInnsyn (fellesløsning)</li><li>Minkommune (fellesløsning)</li></ul> |
| **Samarbeid**<br>*Samhandling på tvers av offentlig og privat forvaltning.* | <ul><li>SKATE-samarbeidet</li><li>Fagarenaer og nettverk</li><li>Internasjonalt samarbeid</li></ul> |
| **Veiledning**<br>*At veiledninger utarbeides og benyttes i felles økosystem.* | <ul><li>Veiledning for sammenh. tjenester</li><li>Beste praksis hendelsesarkitektur</li><li>Nasjonal verktøykasse for deling</li></ul> |
| **Datakilder**<br>*Tilgjengeliggjøre data som nasjonal fellesressurs.* | <ul><li>Folkeregisteret (fellesløsning)</li><li>Enhetsregisteret (fellesløsning)</li><li>Matrikkelen (fellesløsning)</li><li>FIKS Register (fellesløsning)</li></ul> |
| **Datadrevet**<br>*Systematisk bruk av data for innsikt og bedre tjenester.* | <ul><li>data.altinn.no (fellesløsning)</li><li>Analyse av livshendelser</li><li>Strategisk fremsyn basert på data</li></ul> |
| **Standardisering**<br>*Identifisere og fremme standarder for interoperabilitet.* | <ul><li>Referansekatalogen for IT-standarder</li><li>Arkitektur- og standardiseringsrådet</li><li>Rammeverk for digital samhandling</li></ul> |
